# True-Chess Rules

## Core Differences from Traditional Chess

### Piece Representations:
- **Pawns** = Living Bodies (G₁) - Foundation of safety
- **Knights** = Psychological States (G₂) - Unconventional thinking  
- **Bishops** = Data & Memory (G₃) - Diagonal truth perspective
- **Rooks** = Ecosystems (G₄) - Structural integrity
- **Queens** = Future Generations (G₅) - Maximum potential
- **Kings** = Cultural Continuity (G₆) - Core identity
- **Scribe Piece** = Pattern Integrity (G₇) - Game history & rules

### Victory Conditions:
- Not checkmate, but **Harmonic Resolution**
- Achieve board state where all piece types are in balanced positions
- Maximum future options while maintaining safety
